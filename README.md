PROYECTO DE PROGRAMACIÓN - GESTIÓN DE CIUDADES
==============================================

El programa se encarga de leer datos de un fichero CSV e insertarlos en una base de datos, consta de dos menús, uno en el que encontramos la tabla sobre la que podemos hacer operaciones de inserción, borrado y actualizado de datos y otro menú con un formulario en el cual se muestran los datos con botones de avance y retroceso, además de filtros para buscar registros por ciudad y país para mostrarlos sobre el formulario.
